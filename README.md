# To Do List App
The front-end is served on http://localhost:4200/ and the back-end on http://localhost:3001/.

## Setup 
### Angular Setup 

```
cd todo-app 
npm i
ng serve
```

### Node Setup 

```
cd server
npm i
npm start
```

### DB Setup

No db setup required. Usage of sqllite.
Please refer to [sqllitedb.js](server/sqllitedb.js) for database.
